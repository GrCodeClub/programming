      Class

      Εντολή Περιγραφή
      class Ορίζει τη δήλωση μιας κλάσης. Παράδειγμα:
      public class MyClass { }
      public Προσδιορίζει την πρόσβαση της κλάσης. Μια δημόσια κλάση είναι προσβάσιμη από άλλες κλάσεις.
      private private
      protected Προσδιορίζει την πρόσβαση μελών της κλάσης που είναι ορατά στις υποκλάσεις της και στο ίδιο πακέτο.
      static Ορίζει ένα πεδίο ή μέθοδο που ανήκει στην κλάση, αντί να ανήκει σε κάθε αντικείμενο της κλάσης.
      void Ορίζει μια μέθοδο που δεν επιστρέφει κάποια τιμή.
      this Αναφέρεται στο τρέχον αντικείμενο της κλάσης.
      super Αναφέρεται στο γονικό αντικείμενο της κλάσης.
      new ημιουργεί ένα νέο αντικείμενο μιας κλάσης
      Παράδειγμα:
      MyClass obj = new MyClass();
      extends Ορίζει μια κλάση ως υποκλάση μιας άλλης κλάσης
      implements Υλοποιεί μια διεπαφή από μια κλάση
      constructor
