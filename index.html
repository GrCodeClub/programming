      Προγραμματισμός

      Ο προγραμματισμός επιτρέπει στους προγραμματιστές να δημιουργούν λογισμικό, εφαρμογές, ιστοσελίδες και πολλά άλλα. Οι γλώσσες προγραμματισμού μπορούν να είναι γενικού σκοπού, όπως η Java, η C++ και η Python, ή ειδικού σκοπού, όπως η SQL για τη διαχείριση βάσεων δεδομένων ή η HTML/CSS για τη δημιουργία ιστοσελίδων.

      Συντακτικά λάθη

      Ονομάζουμε τα σφάλματα που σχετίζονται με τη σύνταξη μιας γλώσσας προγραμματισμού.

      Αλγόριθμος

      Αλγόριθμο ονομάζουμε ένα σύνολο εντολών (οδηγιών) που, αν εκτελεστούν με ακρίβεια, οδηγούν στην πραγματοποίηση μιας εργασίας ή την επίλυση ενός προβλήματος.

      Γλώσσα υψηλού επιπέδου και Γλώσσα χαμηλού επιπέδου

      Η γλώσσα υψηλού επιπέδου θεωρείται η γλώσσα εκείνη που είναι αρκετά περιγραφική και είναι πιο κοντά στην ανθρώπινη γραπτή γλώσσα. Ενώ γλώσσα χαμηλού επιπέδου θεωρείται εκείνη που είναι πιο κοντά στη μηχανή.

      Μεταβλητή

      Μεταβλητή είναι μια θέση μνήμης στον υπολογιστή την οποία έχουμε δεσμεύσει και της έχουμε ορίσει ένα όνομα. Το περιεχόμενο της συγκεκριμένης θέσης μνήμης είναι η τιμή της συγκεκριμένης μεταβλητής.

      Loop (επαναλήψεις) με χρήση For

      Την εντολή for την χρησιμοποιούμε σε ένα loop όταν γνωρίζουμε τον αριθμό επαναλήψεων.

      Loop (επαναλήψεις) με χρήση While

      Την εντολή while την χρησιμοποιούμε:

      1. όταν θέλουμε να γίνετε ένα loop για όσο ισχύει μια συνθήκη που έχουμε ορίσει.
      2. Μπορούμε να την χρησιμοποιήσουμε ακόμα και όταν γνωρίζουμε τον αριθμό επαναλήψεων. Για να πραγματοποιήσουμε κάνουμε χρήση ενός μετρητή, για να μετράμε τις επαναλήψεις και με μια συνθήκη ελέγχουμε αν έχουμε κάνει τον αριθμό επαναλήψεων που επιθυμούμε. (π.χ. ο μετρητής i (στην αρχή i=0) αυξάνετε στο τέλος κάθε επανάληψης +1 και στο while υπάρχει η συνθήκη (i<9) ώστε να ελέγχει αν έχουν γίνει 9 επαναλήψεις.)

      Κύκλος Ζωής Εφαρμογών

      Μια εφαρμογή ξεκινάει τον κύκλο ζωής της από τη στιγμή που θα καθοριστούν οι απαιτήσεις και οι προδιαγραφές της, και τελειώνει, όταν εξαντληθούν τα περιθώρια συντήρησής της

      1. Ανάλυση
      2. Σχεδίαση
      3. Υλοποίηση
      4. Λειτουργία
      5. Συντήρηση
