      Bιβλιοθήκες

      Βιβλιοθήκη Περιγραφή
      #include <stdio.h> Βιβλιοθήκη εισόδου/εξόδου που παρέχει συναρτήσεις όπως printf() και scanf().
      #include <stdlib.h> Βιβλιοθήκη που παρέχει συναρτήσεις για τη διαχείριση μνήμης, δυναμική ανάθεση μνήμης και άλλες γενικές λειτουργίες.
      #include <math.h> Βιβλιοθήκη μαθηματικών που παρέχει συναρτήσεις όπως το τετραγωνικό ρίζωμα, η εκθετική συνάρτηση, οι τριγωνομετρικές συναρτήσεις και άλλες.
      #include <string.h> Βιβλιοθήκη για λειτουργίες σχετικές με τις συμβολοσειρές, όπως αντιγραφή, συγκατάθεση, σύγκριση και άλλες.
      #include <ctype.h> Βιβλιοθήκη για τον έλεγχο και τη μετατροπή χαρακτήρων, όπως οι συναρτήσεις isalpha(), isdigit() και toupper().
      #include <time.h> Βιβλιοθήκη που παρέχει συναρτήσεις για την εργασία με την χρονολογία και την ώρα, όπως η time() και η strftime().
      #include <stdbool.h> Βιβλιοθήκη που παρέχει τον τύπο δεδομένων bool και τις τιμές true και false για λογικές εκφράσεις.
      #include <limits.h> Περιέχει σταθερές που παρέχουν πληροφορίες για τα όρια των διάφορων τύπων δεδομένων, όπως το μέγιστο και ελάχιστο τιμή.
      #include <assert.h> Χρησιμοποιείται για την υλοποίηση ελέγχων της προϋπόθεσης και της ασφάλειας του προγράμματος.
      #include <stdarg.h> Χρησιμοποιείται για τη δημιουργία συναρτήσεων που δέχονται μεταβλητό αριθμό ορισμάτων.
      #include <signal.h> Παρέχει τις δυνατότητες επικοινωνίας με το λειτουργικό σύστημα μέσω σημάτων.
      #include <errno.h> Χρησιμοποιείται για τη διαχείριση σφαλμάτων που σχετίζονται με λειτουργίες συστήματος.
      #include <pthread.h> Χρησιμοποιείται για τα threads.
      #include <unistd.h> Χρησιμοποιείται για διάφορες λειτουργίες συστήματος, όπως sleep.
      #include <curl/curl.h>
      #include <winsock2.h> Επιτρέπει τη δημιουργία και διαχείριση δικτυακών συνδέσεων (Windows Sockets).
      #include <arpa/inet.h> Περιέχει δηλώσεις για συναρτήσεις που χρησιμοποιούνται για τη μετατροπή διευθύνσεων δικτύου και port από την αναπαράσταση δικτύου στην αναπαράσταση μηχανής και αντίστροφα.
